# Contributor Covenant Code of Conduct

## 我們的承諾

為了營造開放且友善的環境，我們作為貢獻者和維護者承諾：無論年齡、體型、身心障礙、族群、性別認同與表達、經驗程度、教育程度、社經地位、國籍、外貌、種族、宗教或性認同與取向，參與我們專案與社群的每個人都不會受到騷擾。

## 我們的標準

有助於創造正向環境的行為包括：

* 使用友善且包容的語言
* 尊重不同的觀點與經驗
* 優雅地接受建設性批評
* 專注於對社群最有利的事情
* 對其他社群成員展現同理心

不可接受的行為包括：

* 使用性暗示的語言或圖像，以及不受歡迎的性關注或騷擾
* 酸民/挑釁/侮辱性評論，以及人身或政治攻擊
* 公開或私下騷擾
* 未經明確許可，發布他人的私人資訊（如地址或電子郵件）
* 其他在專業場合中被合理認為不適當的行為

## 我們的責任

專案維護者有責任為可接受的行為標準提供說明，並對任何不可接受的行為採取適當且公平的糾正措施。

專案維護者有權且有責任移除、編輯或拒絕不符合本行為準則的評論、提交、程式碼、wiki 編輯、Issue 和其他貢獻，並可暫時或永久禁止任何做出不適當、威脅、冒犯或有害行為的貢獻者。

## 適用範圍

當個人代表專案或社群時，本行為準則適用於專案空間和公共空間。代表專案或社群的範例包括：使用官方專案電子郵件地址、透過官方社群媒體帳號發文，或在線上或線下活動中擔任指定代表。

## 執行

如發現濫用、騷擾或其他不可接受的行為，請向專案團隊回報。所有投訴都將被審查和調查，並將做出被認為必要且適當的回應。專案團隊有義務對事件回報者保密。
