行動建議

  • 🔴 立即修補 Splunk Enterprise CVE-2026-20253（CISA KEV，截止 2026-06-21）。隔離 PostgreSQL 側車服務端口，確認 Splunk 未暴露於公開網際網路。
  • 🔴 稽核所有 Fortinet FortiGate / FortiOS 設備：更新至最新 FortiOS 版本、停用外部暴露管理介面、啟用多因素驗證、輪換所有管理員憑證。台灣企業尤需優先處理（全球第三大受影響國）。
  • 🔴 修補 Joomla Content Editor CVE-2026-48907（CISA KEV，截止 2026-06-19）。稽核所有使用 Joomla 的公開網站，禁用或更新受影響套件。
  • 🟡 更新 Spring AI 至 1.0.9（1.0.x 分支）或 1.1.8（1.1.x 分支），修補 CVE-2026-47835 SQL 注入漏洞（影響 AI 向量資料庫元件）。
  • 🟡 盤點組織內 AI 代理人（Copilot、自建 Agent）的存取範圍與系統權限，實施最小權限原則，針對提示注入攻擊（AutoJack、M365 Copilot）部署行為監控。
  • 🟡 強化 EDR 的防篡改保護（Tamper Protection），部署多層防禦架構，監控安全程序的異常終止事件，應對 GentleKiller 等 EDR 終結工具威脅。
  • 🟢 更新 Adobe Acrobat Chrome 擴充功能至 26.5.2.3 以上，修補 CVE-2026-48294 跨站腳本漏洞，防止 session 資料洩露。
